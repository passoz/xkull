# Decisões

Log de auditoria dos ajustes feitos no plano. Cada arquivo é **imutável** — se mudar de ideia, cria outro.

## Convenção de nome

`AAAA-MM-DD-titulo-curto.md`

Exemplos:
- `2026-06-14-ajuste-corte-150kcal-plato.md`
- `2026-07-07-decisao-injetavel.md`
- `2026-10-22-inicio-reverse-diet.md`

## Template

```markdown
# Decisão — AAAA-MM-DD

## Contexto
- Fase atual: ___
- Peso médio 7d: ___ kg
- Aderência últimos 14 dias: ___%

## Mudança
- **De**: ___
- **Para**: ___

## Motivo
[explicação curta do gatilho]

## Critério de reversão
[quando voltar atrás, se não funcionar]
```
